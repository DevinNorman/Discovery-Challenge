Challenge 1: Sighting data visualisation


1) Collect information through API.

Upon receiving the test I attempted to download the required data from a browser. This did not work and I then attempted to access the data programmatically using python. This was pretty challenging because I have not used pyhton since university so I needed to do a bit of self study and because I was struggling to download the required data I felt like I was doing something wrong. Again this did not work and I attempted using a nuget package and dotnet core. Eventually figuring out that there was a problem on Amazon I was sent the required data and manually added it to my solution.


2) Wrangle data.

In order to use the data I created an angular component to read a CSV file and converted it into a JSON object. I created data models to map the CSV file so that I could use them later when populating the heat map. I then uploaded the object onto Google Firestore to use on my dashboard component. Which is responsible for loading the heat map.


3) Plot on any mapping tool to create a heat map.

In order to plot the data I initially looked at using d3.js and didn't find anything usable. I then looked at using Google Maps and found that I needed to enable the Google Maps API. I found a nice example here.


4) Write up any challenges you faced, how you approached the problem and any insights from the data.

An interesting problem I faced when trying to solve the challenge was the fact that the coordinates had a lot of decimal points. This meant that when I plotted the heat map it did not come out very clearly. In order to solve this I created a hash map and rounded the coordinates to have less decimals. If the coordinates already existed then I gave them a weight so that information would not be lost.

If you would like to look at the code please click here.

The heat map with less items had hot spots around Fairmont Ridge, Fairvale and a little bit around Sandton. The heat map with more hot spots had hot spots around Sandton and at Brentwood estate. I also noticed the heat map followed more along the lines of the main roads. I would have also liked to create a heat map and give weight to the spots by speed rather that just number of coordinates in that area. This would show locations with the most movement which I suspect would be roads rather than where a device is stationary such as home or office.
